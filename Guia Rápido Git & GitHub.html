Guia Rápido Git & GitHub

1. Montou um código novo? Subir para o GitHub

No GitHub: crie um repositório vazio (sem README, .gitignore ou license).

cd caminho/do/projeto
git init -b main
git remote add origin https://github.com/seu-usuario/seu-repo.git
git add .
git commit -m "Primeiro commit"
git push -u origin main

2. Atualizações diárias

Após fazer mudanças:

git add .
git commit -m "Descrição curta do que mudou"
git push

3. Criar ou mudar de branch

Para criar uma nova branch e mudar para ela:

git checkout -b nome-da-branch
# ou
git switch -c nome-da-branch

Para mudar para uma branch existente:

git checkout nome-da-branch
# ou
git switch nome-da-branch

4. Verificar branches

Listar branches locais:

git branch

Listar branches remotas:

git branch -r

Listar todas (local + remota):

git branch -a

5. Deletar branches

Deletar branch local (seguro):

git branch -d nome-da-branch

Forçar deleção:

git branch -D nome-da-branch

Deletar branch remota:

git push origin --delete nome-da-branch

Comandos Avançados

6. Clonar um repositório

git clone https://github.com/usuario/repo.git

7. Ver o estado atual & diferenças

Verificar o estado:

git status

Ver diferenças não comitadas:

git diff

8. Atualizar do remoto

git pull

9. Mesclar e resolver conflitos

git merge outra-branch
# editar conflitos, depois:
  git add .
  git commit

10. Stash (guardar mudanças temporárias)

git stash     # guarda alterações

git stash pop # restaura alterações

11. Histórico (log)

git log --oneline --graph --all

12. Reverter commits

Criar commit de desfazer:

git revert <hash>

Resetar posição do HEAD (cuidado!):

git reset --hard <hash>

13. Tags

git tag v1.0.0
git push origin v1.0.0
